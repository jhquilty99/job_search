# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## What This Repo Is

A Claude Code-powered job search pipeline for John Hayden Quilty. There is no build system, no tests, and no code — the "application" is a set of Claude skills that search the web for job postings, score them, and save ranked results as Markdown files.

## Key Files

| File | Purpose |
|------|---------|
| `profile.yml` | Single source of truth for the candidate's identity, target roles, superpowers, and compensation floor ($160k+) |
| `prospect.md` | Search criteria: role titles, must-haves, deal-breakers, keywords, and target industries |
| `research_output.md` | Generated by `/researcher` — raw list of 15–25 job postings |
| `top5_matches.md` | Generated by `/summarizer` — top 5 ranked matches with scored breakdown |

## Skills (Slash Commands)

| Command | What it does |
|---------|-------------|
| `/orchestrator` | Runs the full pipeline end-to-end (researcher → summarizer) |
| `/researcher` | Searches the web for job postings, writes `research_output.md` |
| `/summarizer` | Scores all jobs in `research_output.md` and writes `top5_matches.md` |

All skills are defined in `.claude/skills/*/SKILL.md`.

## Scoring Model

The summarizer scores every job across 6 weighted dimensions:

| Dimension | Weight |
|-----------|--------|
| Role Title Fit | 25% |
| Tech Stack Match | 25% |
| Compensation | 20% |
| Remote/Location | 15% |
| Company Signal | 10% |
| Growth Potential | 5% |

Jobs are auto-disqualified for deal-breakers (pure analytics, below $160k base, requires relocation, contract-only). On ties, higher Tech Stack Match wins.

## Updating the Candidate Profile

- Edit `profile.yml` to change roles, compensation floor, superpowers, or contact info.
- Edit `prospect.md` to update target companies, deal-breakers, or search keywords.
- Both files are read fresh on every skill run — no cache to invalidate.
